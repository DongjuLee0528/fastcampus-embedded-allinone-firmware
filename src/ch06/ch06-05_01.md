# Timer/Counter - CTC Mode OCM Interrupt 정리

## 1. Timer/Counter 개념
Timer/Counter는 마이크로컨트롤러에서 시간 측정, 주기 생성, 인터럽트 발생 등을 위해 사용하는 하드웨어 모듈이다.

대표적인 활용
- 일정 시간 간격 인터럽트 발생
- 주기적인 신호 생성
- PWM 생성
- 시간 측정


---

# 2. CTC Mode (Compare Timer on Compare Match)

CTC 모드는 타이머 값을 특정 값(OCR)과 비교하여 타이머를 자동으로 초기화하는 방식이다.

## 동작 방식

1. 타이머 카운터(TCNT)가 0부터 증가한다.
2. TCNT 값이 OCR 값과 같아지면 Compare Match가 발생한다.
3. 타이머는 자동으로 0으로 초기화된다.
4. 다시 카운트가 반복된다.

즉 다음과 같은 구조로 반복된다.

```
0 → 1 → 2 → ... → OCR → 0 → 반복
```

## 특징

- 상향 카운터 방식
- 0x00 ~ OCR 값 범위를 반복
- TCNT와 OCR 값이 같아지면 Clear
- 해당 순간 인터럽트 발생 가능




---

# 3. OCM Interrupt (Output Compare Match Interrupt)

OCM 인터럽트는 TCNT 값과 OCR 값이 일치하는 순간 발생하는 인터럽트이다.

## 발생 조건

```
TCNT == OCR
```

이 순간 다음이 발생한다.

- Compare Match 발생
- 인터럽트 트리거


## 인터럽트 벡터

ATmega 계열에서는 다음과 같이 정의된다.

```
TIMERx_COMPA
```

예시

```
TIMER0_COMPA
TIMER1_COMPA
TIMER2_COMPA
```

이 인터럽트가 발생하면 해당 ISR(Interrupt Service Routine)이 실행된다.
